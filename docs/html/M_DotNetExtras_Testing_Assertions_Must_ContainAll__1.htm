      MustContainAllT(IEnumerableT, Boolean) Method

      Asserts that all expected values exist in a collection.

      Definition

      Namespace: DotNetExtras.Testing.Assertions
      Assembly: DotNetExtras.Testing (in DotNetExtras.Testing.dll) Version: 1.0.0
      C#
      public Must ContainAll<T>(
	IEnumerable<T>? expected,
	bool partial = false
)

      Parameters

      expected  IEnumerableT
      Collection of expected values.
      partial  Boolean  (Optional)
      For complex types, indicates whether the missing or null properties in the expected value must be ignored in the actual value.

      Type Parameters

      T
      Type of elements in the collection.

      Return Value

      Must
      The current Must instance.

      Remarks

      Expected: (a, b, c).ContainAll(a, b): true; (a, b, c).ContainAll(c, d): false; (a, b, c).ContainAll(d, e): false.

      Example

      Check a collection of objects.
      C#
      List<User> users =
[
    new() { Name = new() { GivenName = "Joe", Surname = "Black" } , Id = "00000" },
    new() { Name = new() { GivenName = "Mary", Surname = "Beth" } , Id = "54321" },
    new() { Name = new() { GivenName = "John", Surname = "Doe" } , Id = "12345" },
    new() { Name = new() { GivenName = "Jack", Surname = "Green" } , Id = "13579" },
];

users.Must().ContainAll<User>
([
    new() { Name = new() { GivenName = "John", Surname = "Doe" }, Id = "12345" }, 
    new() { Name = new() { GivenName = "Jack", Surname = "Green" } , Id = "13579" }
], false);

users.Must().ContainAll<User>
([
    new() { Name = new() { GivenName = "John", Surname = "Doe" } },
    new() { Name = new() { GivenName = "Jack", Surname = "Green" } }
 ], true);
